 Ocean Expense Tracker – Flutter App

A simple and responsive **Flutter mobile app** to track daily expenses. This version is beginner-friendly and showcases Flutter UI skills, including gradients, responsive layouts, and state management.

Features
- Ocean night gradient background for a clean, themed look 🌊
- Add and delete expenses
- Expense categories with icons (Food, Transport, Shopping, Bills, Others)
- Display total expenses dynamically
- Responsive layout for different screen sizes


  Prerequisites
- Flutter SDK installed
- Dart SDK
- A device or emulator for running the app/ Web
